# 🍽️ DineHub - Console-Based Food Ordering & Delivery System

DineHub is a **Java console-based food ordering and delivery application** that simulates the workflow of an online food delivery platform. The project is built using **Core Java**, follows **Object-Oriented Programming (OOP)** principles, and leverages the **Java Collections Framework** to create a modular and menu-driven application.

The system supports multiple user roles, including **Admin**, **Customer**, **Restaurant Manager**, and **Delivery Person**, each with dedicated functionalities to provide a realistic food ordering and delivery experience.
